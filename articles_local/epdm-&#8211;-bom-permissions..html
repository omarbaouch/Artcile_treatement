When creating a  BOM, it is obvious that permissions need to be assigned in the  Bill of Material – BOM dialogue box.

 

What is often forgotten is to give the User/Group permissions to a BOM.

 

Permissions should be set at the vault root folder. Both Activate computed BOM and See computed BOM permissions can be set.

If permissions are not set for either of the two, above locations, the user will receive a message “Please create at least one bill of materials type column set before using this function. Column sets can be set up in the Administration Tool”, when accessing a file in the vault.
