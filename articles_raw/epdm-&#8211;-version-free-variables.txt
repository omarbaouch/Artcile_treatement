Version free variables allow for a variable to be changed without checking out a file.
Changing a version free variable will not increment the version of a file and a user can update the variable without having permissions to check out the file. This can be useful when their are values that need to be updated, but the actual file does not change. An example of this type of variable would be cost. Version free variables is an option when a variable is created or edited.
