Here are the steps to reset
SOLIDWORKS EPDM
admin password if it is lost or forgotten.
Login into the archive server, as a local system administrator.
Open Windows Regedit
Delete Admin in the key HKEY_LOCAL_MACHINE\SOFTWARE\SolidWorks\Applications\PDMWorks Enterprise\ArchiveServer\ConisioUser.
Delete Admin Key
Delete Admin in the key HKEY_LOCAL_MACHINE\SOFTWARE\SolidWorks\Applications\PDMWorks Enterprise\ArchiveServer\Vaults\[vault]\ConisioUsers.
Open Archive server Configuration tool.
Access Archive Server Configuration from Start
Right click on the vault and click on properties.
Access vault properties
Under the Login tab select Password.
Login Properties
Enter your current and new password (twice) in the Admin Password dialog to reset SOLIDWORKS EPDM admin password
Enter a New Password
