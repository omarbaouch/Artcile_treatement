If you need to change the SOLIDWORKS PDM user password, you will need access to the SOLIDWORKS PDM Administration Tool.

  1. Login as ‘admin’ in the Administration Tool
  2. Expand the ‘Users and Groups Management’
  3. Expand ‘Users’
  4. Right-click on the user profile and select ‘Change Password…’

    Users and Group Management

  5. Enter the new password and confirm the new password in the change password window. You will also need to enter the password of the logged-in user (admin) and select OK.

    Change Password
