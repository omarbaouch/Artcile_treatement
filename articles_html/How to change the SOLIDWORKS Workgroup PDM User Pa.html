If you need to change your SOLIDWORKS Workgroup PDM user password, this can be done directly within the SOLIDWORKS add in for Workgroup PDM.

Right click within the Workgroup PDM tab in the Task Pane and select “SOLIDWORKS Workgroup PDM Options…”

SOLIDWORKS Workgroup PDM Options

Go to the “Vault” tab, under the “Password” section click “Change” to modify the existing serial number.

Workgroup Vault Options
