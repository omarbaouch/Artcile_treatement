If you need to change the
SOLIDWORKS PDM
user password, you will need access to the SOLIDWORKS PDM Administration Tool.
Login as ‘admin’ in the Administration Tool
Expand the ‘Users and Groups Management’
Expand ‘Users’
Right-click on the user profile and select ‘Change Password…’
Users and Group Management
Enter the new password and confirm the new password in the change password window. You will also need to enter the password of the logged-in user (admin) and select OK.
