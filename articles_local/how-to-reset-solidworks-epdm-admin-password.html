Here are the steps to reset SOLIDWORKS EPDM admin password if it is lost or forgotten.

  1. Login into the archive server, as a local system administrator.
  2. Open Windows Regedit
  3. Delete Admin in the key HKEY_LOCAL_MACHINE\SOFTWARE\SolidWorks\Applications\PDMWorks Enterprise\ArchiveServer\ConisioUser.

Delete Admin Key

  1. Delete Admin in the key HKEY_LOCAL_MACHINE\SOFTWARE\SolidWorks\Applications\PDMWorks Enterprise\ArchiveServer\Vaults\[vault]\ConisioUsers.

  1. Open Archive server Configuration tool.

Access Archive Server Configuration from Start

  1. Right click on the vault and click on properties.

Access vault properties

  1. Under the Login tab select Password.

Login Properties

  1. Enter your current and new password (twice) in the Admin Password dialog to reset SOLIDWORKS EPDM admin password
