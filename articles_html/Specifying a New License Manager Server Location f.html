When server components are moved, particularly in the SOLIDWORKS Solid Network License Manager, the new server location must be updated to ensure continued access to licenses. This update is performed within the SOLIDWORKS PDM Administration tool.

Updating the License Manager Server in SOLIDWORKS PDM

To specify the new license server location, follow these steps:

  1. Open SOLIDWORKS PDM Administration.
  2. Double-click on the License node to open the license settings.
  3. In the Server List, add the new server and remove the old one.
  4. Specify the new server using the format: port@server_name.
    • The default port for the SOLIDWORKS SolidNetWork License Manager is 25734.

Monitoring License Usage

In addition to updating the license server location, license usage can be monitored from the License node within SOLIDWORKS PDM Administration. To do this:

  1. Navigate to the License Usage tab within the License node.
  2. Review active sessions, allocated licenses, and user activity.

License Usage

By ensuring that the SOLIDWORKS Solid Network License Manager server location is correctly defined and monitoring license usage, administrators can maintain a seamless licensing experience for SOLIDWORKS PDM users.
